# Clawcoin 設計規範 v2.0
**更新日期：2026.03.21**

---

## 🦞 核心視覺：紅色龍蝦機器人

### 角色設定

```
名稱：SHROMBY（蝦龍比）
物種：機械改造龍蝦
顏色：紅色（#FF4444 / #E63946）
特徵：
  - 鈎型大爪子（左手抓比特幣）
  - 機器人眼睛（發光藍色LED）
  - 機械關節外露
  - 龍蝦尾扇為渦輪風扇
  - 頭戴數據頭盔（HUD顯示）
```

### 姿態設計

```
主視角：
     ◄►
   [===]  ← 機器人眼睛（發光藍光）
  /    \  ← 機械觸角
 [蝦殼身]  ← 紅色金屬蝦殼
  |  |   ← 機械關節
 [鉗BTC]  ← 左手：金色機械爪子抓著 ₿
  \__/   ← 右手：攻擊姿態，准戰鬥
   ||    ← 龍蝦尾（渦輪加速模式）
```
