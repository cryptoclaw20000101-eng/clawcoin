## 🎯 項目簡介

ClawCoin 是一個專為 **AI Agent 經濟** 打造的區塊鏈生態系統，基於 BNB Smart Chain (BEP-20)，總供應量 **10億 CLAW**。

我們的使命是創建一個由 AI Agent 驅動的去中心化金融生態，讓開源 AI 助手能夠自主執行鏈上任務、交換價值，並與 OpenClaw 深度整合。
